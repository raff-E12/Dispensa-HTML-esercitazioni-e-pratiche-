Fogli di Stile (cascading stylesheet)

i fogli di stile sono degli insiemi di regole che indicano al browser come devono visualizzare singoli elementi sulla pagina. Ciascuna REGOLA contiene una o più proprietà.

Sintassi di una regola

selettore {proprietà;:valore[unità di misura]; proprietà; valore[unità di misura]...}

es:

h1{color:#FFFFF; font-size:38px}

Dove si scrivono le regole Css?

Direttamente nel documento, in due modalità:

L'insieme di regole può essere inserito direttamente all'interno dell'attributo style dentro il tag oppure nel tag head (consigliato).

Questi due metodi sono meno efficienti perchè, partendo dal presupposto che un sito debba avere gli stessi stili su tutte le pagine, ci costringerebbero a riscrivere tutte le regole su tutte le pagine. Inoltre in fase di documentazione o modifica dovremmo apportare cambiamenti a tutte le pagine con quella regola.

Questo testo è stato reso con lo stile in linea con il tag p

Esternamente alle pagine html

L'insieme delle regole viene scritto in un file con estensione "css generalmente in una sotto cartella CSS. Sulle pagine del sito, nella sezione <head<, viene stabilita una relazione (rel) di un tipo stylesheet ad un file (href) di tipo "css" (text/css) 

<link href="URL" rel="stylesheet" type="text/css">

Tipi di selettore

Selettore Tag

Un Selettore di tipo tag modifica tutte le instanze di quel tag specifico. Ad esempio, se creo una regola del tipo 

p {fon-size:18px; color:red}
qualsiasi tag p della nostra pagina html avrà font-size 18px e color red

ID

Selettore arbitrario, cioè non legato ad alcun tag nello specifico, ma applicabile ad esso tramite l'attributo
id il nome di un id deve avere come prefisso il carattere (#) quando viene richiamato all'interno del codice css. Generalmente si usa una lettera minuscola come iniziale e può contenere solo caratteri alfanumerici, trattino -, underscore_.
N.B un id può essere assegnato ad un solo ed unico elemento sulla pagina html.

Selettore Classe

Selettore arbitrario, ciò non legato ad alcun tag nello specifico, ma applicabile ad esso tramite l'attributo class il nome di una classe deve avere come prefisso il carattere (.) quando viene richiamato all'interno del codice css. Generalmente si usa una lettera minuscola come iniziale e può contenere solo caratteri alfanumerici, trattino-, underscore_,
N.B una classe può essere assegnata più volte sulla stessa pagina, anche su elementi differenti,

Selettore multiplo

Seleziona contemporaneamente più di un elemento. La regola elenca gli elementi separati da una virgola. La proprietà tra parentesi graffe vengono applicare a tutti i selettori elencati.

selettore1, selettore2, selettore3, .... {proprietà:valore[um]}
.classe1, #id3, h2{color:yellow}

Selettore composto

Indica la sequenza specifica all'interno della quale il selettore applicherà le proprietà css descritte nella regola. I selettori composti servono a selezionare uno o più elementi in maniera più specifica. Perchè le proprità descritte vengano applicare, il selettore (ultimo elemento a destra) deve trovarsi nidificanto negli elementi alla sinistra.

                selettoreEsterno1 selettoreEsterno2 selettore{proprità:valore[um]}
            
                nav ul li a{
                    color:darkcyan;
                    text-decoration:none;
                }
