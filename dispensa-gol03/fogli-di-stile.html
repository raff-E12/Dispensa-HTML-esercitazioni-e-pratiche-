I fogli di stile (cascading stylesheet)

I fogli di stile sono degli insiemi di regole che indicano al browser come visualizzare singoli elementi sulla pagina. Ciascuna REGOLA contiene una o più proprietà.

Sintassi di una regola

selettore{proprietà:valore[unità di misura]; proprietà:valore[unità di misura];...}

es:

h1{color:#ff0000; font-size:28px}

Dove si scrivono le regole css?

Direttamente nel documento, in due modalità

L'insieme di regole può essere inserito direttamente all'interno dell'attributo style dentro il tag oppure nel tag style, dentro il tag head (consigliato).

Questi due metodi sono meno efficienti perché,partendo dal presupposto che un sito debba avere gli stessi stili su tutte le pagine, ci costringerebbero a riscrivere tutte le regole su tutte le pagine. Inoltre in fase di manutenzione o modifica dovremmo apportare cambiamenti a tutte le pagine con quella regola.

Si utilizza esclusivamente quando una particolare pagina ha bisogno di una modifica specifica rispetto alle altre.

Questo testo è stato reso rosso con lo stile in linea sul tag p

Esternamente alle pagine html

L'insieme delle regole viene scritto in un file con estensione "css" generalmente in una sottocartella CSS. Sulle pagine del sito, nella sezione <head>, viene stabilita una relazione (rel) di tipo stylesheet ad un file (href) di tipo "css" (text/css)

<link href='URL' rel='stylesheet' type='text/css'>

Tipi di selettore

Selettore Tag

Un selettore di tipo tag modifica tutte le istanze di quel tag specifico. Ad esempio, se creo una regola del tipo 

p{font-size:18px; color:red;}
, qualsiasi tag p della nostra pagina html avrà fonst-size 18px e color red.

ID

Selettore arbitrario, cioè non legato ad alcun tag nello specifico, ma applicabile ad esso tramite l'attributo id. Il nome di un id deve avere come prefisso il carattere (#) quando viene richiamato all'interno del codice css. Generalmente si usa una lettera minuscola come iniziale e può contenere solo caratteri alfanumerici, trattino -, underscore _.
N.B. un id può essere assegnato ad un solo ed unico elemento sulla pagina html.

Selettore Classe

Selettore arbitrario, cioè non legato ad alcun tag nello specifico, ma applicabile ad esso tramite l'attributo class. Il nome di una classe deve avere come prefisso il carattere (.) quando viene richiamato all'interno del codice css. Generalmente si usa una lettera minuscola come iniziale e può contenere solo caratteri alfanumerici, trattino -, underscore _.
N.B. una classe può essere assegnata più volte sulla stessa pagina, anche su elementi differenti.

Selettore multiplo

Seleziona contemporaneamente più di un elemento. La regola elenca gli elementi selezionati separati da una virgola. Le proprietà tra parentesi graffe vengono applicare a tutti i selettori elencati.

selettore1, selettore2, selettore3, ... {proprietà:valore[um]}
.classe1, #id3, h2{color:yellow}

Selettore composto

Indica la sequenza specifica all'interno della quale il selettore applicherà le proprietà css descritte nella regola. I selettori composti servono a selezionare uno o più elementi in maniera più specifica.Perché le proprietà descritte vengano applicate, il selettore (ultimo elemento a destra) deve trovarsi nidificato negli elementi alla sua sinistra.

                selettoreEsterno1 selettoreEsterno2 selettore{proprietà:valore[um]}
            
                nav ul li a{
                    color:darkcyan;
                    text-decoration:none;
                }
