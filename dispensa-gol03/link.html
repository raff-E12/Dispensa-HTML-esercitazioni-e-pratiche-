L'url

Il cardine del sistema internet è che non esistono due risorse che abbiano lo stesso indirizzo. Un indirizzo è il posto fisico in cui si trova una risorsa (una cartella, un'immagine, una pagina web). Gli indirizzi web si chiamano URL (uniform resource locator).

Schema di un URL

Un indirizzo puòpresentarsi in tante forme tra cui:

  • https://www.miodominio.est/path/risorsa.html
  • https://www.miodominio.est/path/immagine.est
  • https://www.sottodominio.miodominio.est/path/risorsa.html

Parti di un url

  • https, http, ftp, ftps, rappresentano lo schema del collegamento, indicano il protocollo che richiameranno ed il tipo di server che risponderà alla richiesta. Http sta per Hyper Text Transfer Protocol
  • www. indica che ci troviamo all'interno del world wide web, può o meno essere presente (dipende dalla configurazione del server)
  • sottodominio - èil dominio di terzo livello, opzionale, non sempre presente (https://www.SOTTODOMINIO.dominio.est)
  • dominio - anche detto dominio di secondo livello (https://www.sottodominio.DOMINIO.est)
  • est - indica il dominio di primo livello o anche detto estensione (https://www.sottodominio.dominio.EST)
  • path - ` il percorso interno al dominio nel quale si trova la risorsa, generalmente una sottocartella (https://www.sottodominio.dominio.est/PATH/pagina.html)
  • risorsa.html - indica la pagina interna alla quale ci dirotterà il browser (https://www.sottodominio.dominio.est/path/RISORSA.HTML)

Attributi e vari tipi di collegamento

Un link nell'html è composto dal tag <a href="URL">testo del link</a>. Questo puòraccogliere al suo interno alcuni attributi che estendono le sue proprietà

Attributi

href, title, target sono gli attributi da poter utilizzare all'interno del tag a, per descrivere al meglio il link.

  • href - indirizzo di destinazione (URL)
  • title - spiega il senso del collegamento, ` per descrivere il contenuto della risorsa ed aiuta in ottica seo e accessibilità
  • target - indica dove si aprirà la risorsa dell'attributo href

Collegamenti relativi

Il collegamento relativo ` un tipo di link che punta ad un URL presente all'interno dello dominio. In questo caso nell'href viene indicato solo il nome della risorsa o della pagina verso la quale punta.

Collegamenti assoluti

Il collegamento assoluto è un tipo di link che punta ad un URL presente all'esterno del nostro dominio. Nell'href deve necesseriamente essere riportato tutto il collegamento per esteso completo di protocollo.
